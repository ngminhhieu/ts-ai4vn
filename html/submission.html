Hướng dẫn nộp kết quả.

Với mỗi file tên gt_{folder_number}_{file_number}.csv, đội thi cần xuất kết ra file .csv. Tên file .csv sẽ có định dạng res_{folder_number}_{file_number}.csv trong đó folder_number là thứ tự của folder, file_number là thứ tự của file trong thư mục tương ứng.

Ví dụ: file gt_1_1.csv thì cần xuất kết quả ra file res_1_1.csv

Sau đó tất cả file res_{folder_number}_{file_number}.csv trong các thư mục có tên dựa theo folder_number tương ứng, tất cả các thư mục sẽ được nén thành file prediction.zip và nộp lên thệ thống.
