Hướng dẫn nộp kết quả

Với mỗi file tên input_{num}.csv, đội thi cần xuất kết ra file .csv tương ứng. File kết quả sẽ có định dạng res_{num}.csv trong đó num là số thứ tự của file.

Ví dụ: file input_1.csv thì cần xuất kết quả ra file res_1.csv. Trong file res_1.csv, kết quả dự đoán được trình bày dạng cột và không yêu cầu có header.
Ví dụ kết quả dự đoán cho 7 giờ tương lai là 40.3, 39.8, 45.6, 52.4, 60.1, 39.8, 44.0, ta cần để các số liệu này vào cột đầu tiên của file mà không có header.

Sau đó tất cả file res_{num}.csv sẽ được nén thành file prediction.zip và nộp lên thệ thống.
